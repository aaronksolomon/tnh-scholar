# Command Line Tools

This directory contains user guides for TNH Scholar's command-line tools.

## Overview

See [overview.md](overview.md) for a comprehensive introduction to all CLI tools.

## Available Tools

- **[audio-transcribe.md](audio-transcribe.md)** - Transcribe audio files with speaker diarization
- **[nfmt.md](nfmt.md)** - Format and normalize text files
- **[tnh-fab.md](tnh-fab.md)** - Fabricate and generate content
- **[tnh-setup.md](tnh-setup.md)** - Set up and initialize TNH Scholar
- **[token-count.md](token-count.md)** - Count tokens in text files
- **[ytt-fetch.md](ytt-fetch.md)** - Fetch YouTube transcripts

## Usage

Each tool guide includes:
- Purpose and use cases
- Command syntax and options
- Examples
- Common workflows

For quick reference, see the main [CLI Overview](overview.md).
