# Coding Prompts

Reusable prompts for TNH Scholar development and repository-maintenance work.

These prompts are intentionally separate from `/prompts`, which is used for TNH Scholar language-processing and research workflows.

Current contents:

- `release_consistency_review.md` - high-level release-facing docs consistency review
- `docs_language_review.md` - editorial/tone review for documentation changes
- `adr_review.md` - bounded ADR implementation-risk review
- `ack_exact.md` - golden smoke-test prompt that must return exact `ACK`

These were promoted from working artifacts in `/tmp` so they can be reused and versioned in-repo.
